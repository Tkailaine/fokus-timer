# Fokus Timer

Aplicação web de temporizador inspirada na técnica **Pomodoro**, desenvolvida para praticar **manipulação do DOM e gerenciamento de estado em JavaScript puro**.

O projeto simula ciclos de foco e pausas para estudo ou trabalho, permitindo alternar entre diferentes modos de produtividade e personalizar o tempo do temporizador.

## Funcionalidades

- Alternância entre **modo foco**, **pausa curta** e **pausa longa**
- Temporizador com **contagem regressiva**
- Controle de **iniciar e pausar** o temporizador
- Música ambiente opcional durante o foco
- Configuração personalizada do tempo de foco
- Painel de configurações com animação suave
- Feedback visual com mudança de contexto e interface

---

## Conceitos aplicados

Este projeto foi desenvolvido para praticar conceitos fundamentais de front-end:

- Manipulação do **DOM**
- **Event listeners** e controle de interações
- **Timers** com `setInterval` e `clearInterval`
- Controle de **estado da interface**
- Atualização dinâmica de conteúdo na página
- Estruturação de lógica em JavaScript
