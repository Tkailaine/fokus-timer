# Fokus Timer

Aplicação web de temporizador inspirada na técnica **Pomodoro**, com gerenciamento de tarefas e persistência de dados no navegador.

O projeto simula ciclos de foco e pausas para estudo ou trabalho, permitindo organizar tarefas durante o uso e manter o estado da aplicação sincronizado com a interface.

## Funcionalidades

- Alternância entre **modo foco**, **pausa curta** e **pausa longa**
- Temporizador com **contagem regressiva**
- Controle de **iniciar e pausar**
- Música ambiente opcional durante o foco
- Configuração personalizada do tempo
- Painel de configurações com animação suave
- Lista de tarefas integrada ao fluxo do app
- Adição, edição e remoção de tarefas
- Persistência de dados com **localStorage**
- Atualização dinâmica da interface conforme estado da aplicação

---

## Conceitos aplicados

- Manipulação do **DOM**
- Gerenciamento de **estado no front-end**
- Persistência de dados com **localStorage**
- Eventos e interações do usuário
- Controle de tempo com `setInterval` e `clearInterval`
- Sincronização entre estado e interface
